# 📦 Tech Supply Chain & Inventory Analytics Dashboard

An interactive Power BI dashboard designed for supply chain, logistics, and inventory analytics. This project monitors product performance, isolates supply chain bottlenecks, and evaluates overall operational efficiency to streamline distribution networks and optimize stock management.

---

## 📌 Project Overview
Efficient logistics and inventory control are essential for sustaining modern hardware and tech distribution channels. This repository features an end-to-end business intelligence pipeline that tracks supply chain operations. 

By evaluating product stock velocity, fulfillment lead times, and operational performance metrics, this dashboard equips supply chain managers with actionable insights to reduce carrying costs and avoid stockouts.

---

## 📂 Repository Structure
As shown in the project repository structure , the files are organized as follows:

*   **`Data/`**: Contains the source datasets used for logistics and inventory tracking.
*   **`Teck Supply.pbix`**: The master Power BI Desktop application containing the data models, table relationships, and interactive visualization layers.
*   **`Logo.jpg`**: Visual branding assets used to establish a cohesive and professional UI theme.
*   **`dach Supply.png`**: A high-resolution layout preview capturing the core operational interface view.

## 🚀 Key Features & Insights
*   **Operational Efficiency Metrics**: Immediate visibility into core performance figures, shipping turnaround rates, and distribution volumes.
*   **Inventory & Product Performance**: Multi-dimensional matrix tracking top-velocity tech items against slow-moving warehouse stock.
*   **Supply Chain Diagnostics**: Monitoring order fulfillment lifecycle distributions to isolate delivery lags and process bottlenecks.

---

## 🛠️ Tools & Technologies Used
*   **Power BI Desktop**: Structural data modeling, cross-filtering configuration, and corporate dashboard layout design.
*   **Power Query**: Schema cleaning, data type adjustments, and managing multi-table data transformations (ETL).
*   **DAX (Data Analysis Expressions)**: Custom expressions calculating performance percentages, inventory turnover indexes, and time-intelligence operational metrics.
